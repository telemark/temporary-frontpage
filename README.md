# temporary-frontpage

[![Greenkeeper badge](https://badges.greenkeeper.io/telemark/temporary-frontpage.svg)](https://greenkeeper.io/)

Base for temporary frontpage

# Usage

Change index.html to suit your needs.

# Deploy to Now

```
$ npm run now-deploy
```

# Docker

```
$ docker build -t temporary-frontpage .
```
